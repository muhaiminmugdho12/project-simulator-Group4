## Test Title
Verify that a Merchant can issue a partial refund for a returned item.

## Description
Test that a Merchant can locate a completed transaction and process a partial refund, with the refunded amount returning to the original payment source of the user.

## Preconditions

- A completed transaction exists  
- Refund option enabled  

## Dependencies
- Payment gateway available  
- User account linked to payment source  

## Test Steps
1. Open Merchant Dashboard  
2. Locate completed transaction  
3. Select Refund option  
4. Enter partial refund amount  
5. Confirm refund  
6. Process refund
